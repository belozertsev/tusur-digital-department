Блог, где каждый пользователь может:
- просматривать, создавать и удалять свои посты и посты других людей (если расрешено)
- изменять настройки доступа к своей странице (GET, POST, DELETE)

Клиентское приложение - SPA c использованием React.js (хранится бандлом в "blog/templates/static")
Серверная часть - Просто API на Django
